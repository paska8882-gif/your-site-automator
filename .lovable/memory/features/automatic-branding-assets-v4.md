# Memory: features/automatic-branding-assets-v4
Updated: now

Система брендингу (логотип, фавікони) використовує комбінацію ШІ-генерації (Gemini 2.5 Flash Image) на основі ніші сайту та розширену бібліотеку з 35+ SVG-шаблонів. Крім базових фігур, додано тематичні форми: корона, блискавка, хвиля, гори, вогонь, компас, якір, ракета, крапля та сонце. Шаблони розподілені за категоріями (travel, fitness, luxury, tech, maritime тощо) та прив'язані до 30+ стилів макетів для максимальної візуальної відповідності тематиці бізнесу. Генеровані активи (logo.svg, favicon.svg, favicon.ico) автоматично включаються до всіх типів проектів (HTML, PHP, React).

## VIP-режим AI-генерації
Для VIP-запитів (коли присутній `vipPrompt`) активується AI-генерація логотипів через google/gemini-2.5-flash-image. Модель генерує унікальне зображення на основі назви компанії та теми бізнесу. Якщо AI-генерація не вдається, система автоматично використовує fallback на SVG-шаблони.

Ключові функції:
- `generateBranding(config, apiKey)` - асинхронна AI-генерація з fallback
- `generateBrandingSync(config)` - синхронна шаблонна генерація
- `ensureFaviconAndLogoInFilesAsync()` - для VIP-режиму
- `ensureFaviconAndLogoInFiles()` - для звичайного режиму
